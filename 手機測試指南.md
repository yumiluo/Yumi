## 🐛 常見問題解決

### 問題1: 手機無法訪問頁面
**可能原因**:
- 防火牆阻擋
- 路由器設置問題
- 網絡配置問題

**解決方法**:
```bash
# 檢查電腦防火牆
sudo ufw status
# 如果啟用，允許端口
sudo ufw allow 3000
sudo ufw allow 5001
```

### 問題2: 頁面加載很慢
**可能原因**:
- 網絡延遲高
- 設備性能問題

**解決方法**:
- 檢查Wi-Fi信號強度
- 確保設備間距離不要太遠

### 問題3: 同步不準確
**可能原因**:
- 網絡延遲
- 設備時間不同步

**解決方法**:
- 系統會自動補償網絡延遲
- 確保所有設備時間同步
